# LearnSmart Color System

This document outlines the professional color grading applied across the LearnSmart app to ensure visual harmony and reinforce brand identity.

## Primary Color Palette

### Light Mode (Light Sand Theme)
- **Primary:** Sage Green (`#9CAF88`) - Used for actions, highlights, and active states.
- **Background:** Warm Sand (`#F5F1E8`) - Main background and cards.
- **Text Primary:** Charcoal (`#2D2D2D`) - Body text and headings.
- **Text Secondary:** Medium Gray (`#666666`) - Secondary text and hints.
- **Borders/Dividers:** Light Gray (`#E0DDD9`).
- **Success:** Green (`#4CAF50`).
- **Error:** Red (`#E53935`).
- **Warning:** Amber (`#FFC107`).
- **Accent:** Gold (`#FFD700`) - Affiliate buttons and highlights.

### Dark Mode (Dark Forest Theme)
- **Primary:** Sage Green (`#9CAF88`) - Consistent with light mode.
- **Background:** Deep Charcoal (`#1A1A1A`) - Main background.
- **Card Background:** Darker Gray (`#242424`).
- **Text Primary:** Off-white (`#E8E8E8`).
- **Text Secondary:** Light Gray (`#A0A0A0`).
- **Borders/Dividers:** Very Dark Gray (`#3A3A3A`).
- **Success:** Light Green (`#66BB6A`).
- **Error:** Light Red (`#EF5350`).
- **Warning:** Light Amber (`#FFD54F`).
- **Accent:** Gold (`#FFD700`).

### 12+ Mode (Corporate Variant)
- **Primary:** Darker Sage (`#4A6741`) - More serious and professional.
- **Background:** Warm Sand (`#F5F1E8`) - Same as light mode.
- **Text:** Charcoal (`#2D2D2D`).
- **Overall:** Slightly desaturated, corporate feel.
