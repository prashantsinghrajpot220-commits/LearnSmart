# LearnSmart Navigation Design

LearnSmart features a unique and memorable navigation system designed to enhance usability and reinforce brand identity.

## Navigation Elements

### 1. Custom Bottom Tab Bar
- **Glassmorphism:** Frosted glass effect for a modern feel.
- **Wave Animation:** Smooth wave transition when switching between tabs.
- **Floating Effect:** Elevated design with subtle shadows.
- **Active State:** Underline and Sage Green highlights.

### 2. Top Navigation Header
- **Branding:** App logo and "LearnSmart" title.
- **Context:** Displays the current page title (e.g., "Mathematics", "Profile").
- **User Actions:** Quick access to notifications and user profile.
- **Sticky:** Always visible at the top while scrolling.
- **Glassmorphism:** Matches the bottom tab bar.

### 3. Breadcrumb Navigation
- **Hierarchy:** Clear path display (e.g., Home > Class 9 > Mathematics).
- **Interactivity:** Clickable paths for easy back-navigation.
- **Styling:** Sage Green for the current page, subtle dividers.

### 4. Floating Action Button (FAB) - Smarty Chat
- **Smarty AI:** Quick access to the AI assistant from any screen.
- **Animation:** Subtle pulsing or bouncing effect to draw attention.
- **Positioning:** Bottom-right, floating above the navigation bar.

### 5. Progress Indicators
- **Chapter Progress:** Circular rings in Sage Green.
- **Daily Streak:** Animated fire icon.
- **XP Bar:** Gradient fill from Sage Green to Gold.

## Design Principles
- **Smoothness:** Transitions and micro-interactions between 200-300ms.
- **Hierarchy:** Clear differentiation between active and inactive states.
- **Responsive:** Optimized for both mobile devices and web platforms.
- **Consistency:** Uses an 8px grid system for spacing and layout.
